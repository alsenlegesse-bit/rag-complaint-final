# CrediTrust Financial - Complaint Analysis Chatbot

## Project Overview
This project implements a RAG-powered chatbot for analyzing customer complaints at CrediTrust Financial.

## Business Objective
Transform unstructured complaint data into actionable insights for product managers, support teams, and compliance officers.

## Tasks
1. **Task 1**: EDA and Data Preprocessing
2. **Task 2**: Text Chunking and Vector Store Creation
3. **Task 3**: RAG Pipeline Development
4. **Task 4**: Interactive Chat Interface

## Setup Instructions
1. Clone the repository
2. Install requirements: `pip install -r requirements.txt`
3. Download the dataset to `data/raw/complaints.csv`
4. Run Task 1: `python src/task1_eda_preprocessing.py`

## Final Project Submission
Complete RAG-based complaint analysis system with:
- Vector embeddings and FAISS search
- LLM-powered analysis
- Gradio web interface
- Comprehensive documentation

See FINAL_REPORT.md for detailed analysis.
